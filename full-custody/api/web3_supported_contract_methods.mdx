---
title: Get Supported Contract Methods
api: GET /v1/custody/web3_supported_contract_methods/ 
description: "This endpoint returns a JSON response with a list of supported contract methods for a given chain and contract address."
---
#### Request
<ParamField query="chain_code" type="String" required>Chain code of the blockchain.</ParamField>
<ParamField query="contract_address" type="String" required>Address of the contract.</ParamField>

#### Response
<ResponseField name="type" type="String" >Contract Method Type</ResponseField> 
<ResponseField name="method_name" type="String" >Contract Method Name</ResponseField> 
<ResponseField name="method_id" type="String" >Contract Method ID</ResponseField>
<ResponseField name="inputs" type="String" >Contract Method Input Parameters</ResponseField>

<RequestExample>
```python Python
request(
    "GET",

    {

    },
    api_key, api_secret, host
)
```
