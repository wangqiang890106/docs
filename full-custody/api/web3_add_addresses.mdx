---
title: Add Addresses
api: POST /v1/custody/web3_add_addresses/ 
description: "This endpoint allows users to add new addresses to their custody wallet. The endpoint takes in a POST request with the chain code and the number of addresses to be added. The maximum number of addresses that can be added at once is 200. The response is a JSON object containing the newly generated addresses."
---
#### Request 

<ParamField body="chain_code" type="String" required>The chain code of the blockchain network.</ParamField>
<ParamField body="count" type="Int" required>The number of addresses to be added.</ParamField>

#### Response
<ResponseField name="addresses" type="String" >The newly generated addresses.</ResponseField>
<RequestExample>
```python Python
request(
    "POST",
    "/v1/custody/web3_add_addresses/",
    {
        "chain_code": "ETH",
        "count": 3
    },
    api_key, api_secret, host
)
```
