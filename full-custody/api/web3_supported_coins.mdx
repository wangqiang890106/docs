---
title: Get Supported Coins
api: GET /v1/custody/web3_supported_coins/ 
description: "This endpoint returns a JSON response with a list of supported coins for a given chain code. The response includes the coin name, display code, description, decimal, and minimum deposit threshold. It also indicates whether the coin can be deposited or withdrawn, based on the current suspension status."
---
#### Request 

<ParamField query="chain_code" type="String" required>Chain code for which to retrieve supported coins.</ParamField>

#### Response 

<ResponseField name="coin" type="String" >Name of the coin.</ResponseField> 
<ResponseField name="description" type="String" >Description of the coin.</ResponseField> 
<ResponseField name="decimal" type="Int" >Number of decimal places for the coin.</ResponseField> 
<ResponseField name="can_deposit" type="Boolean" >Whether the coin can be deposited.</ResponseField> 
<ResponseField name="can_withdraw" type="Boolean" >Whether the coin can be withdrawn.</ResponseField> 
<ResponseField name="minimum_deposit_threshold" type="String" >Minimum deposit threshold for the coin.</ResponseField>
