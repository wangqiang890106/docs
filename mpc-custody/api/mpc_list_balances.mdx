---
title: Get Balances List
api: GET /v1/custody/mpc/list_balances/ 
description: "This endpoint returns a JSON response with a list of token assets and their balances for a given custody wallet. The response includes the total count of token assets, as well as information about each asset, such as the coin type, chain code, display code, and balance. Additionally, if any of the token assets are NFTs, the response will include information about the NFT, such as the NFT code, token ID, and contract address."
---
#### Request
<ParamField query="coin" type="String" >Coin type of the token asset to retrieve.</ParamField>
<ParamField query="chain_code" type="String" >Chain code of the token asset to retrieve.</ParamField>
<ParamField query="page_index" type="Int" required>Index of the page to retrieve.</ParamField>
<ParamField query="page_length" type="Int" required>Number of items to retrieve per page(max:50)</ParamField>

#### Response
<ResponseField name="total" type="Int" >Total number of entries</ResponseField> 
<ResponseField name="address" type="String" >Address of the token asset.</ResponseField>
<ResponseField name="coin" type="String" >Coin type of the token asset.</ResponseField>
<ResponseField name="chain_code" type="String" >Chain code of the token asset.</ResponseField>
<ResponseField name="display_code" type="String" >Abbreviation (reference only, subject to change.</ResponseField>
<ResponseField name="description" type="String" >Description of the token asset.</ResponseField>
<ResponseField name="balance" type="String" >Balance of the token asset.</ResponseField>
<ResponseField name="decimal" type="Int" >Decimal places of the token asset.</ResponseField>
