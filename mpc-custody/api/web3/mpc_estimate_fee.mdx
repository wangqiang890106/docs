---
title: Get Estimate Fee
api: GET /v1/custody/mpc/estimate_fee/ 
description: "This endpoint estimates the fee for a given transaction. It takes in various parameters such as the coin type, amount, address, gas price, gas limit, and extra parameters. The response includes the fee amount in different levels such as slow, average, and fast."
---
#### Request
<ParamField query="coin" type="String" required>coin name</ParamField>
<ParamField query="amount" type="Int" >transaction amount in decimal places (e.g. one bitcoin is divisible to eight decimal places, and 100000000 represents 1 BTC)</ParamField>
<ParamField query="address" type="String" >deposit address used for account model</ParamField>
<ParamField query="replace_cobo_id" type="String" >Cobo ID of a transaction designated as RBF</ParamField>
<ParamField query="from_address" type="String" >from address used for account model</ParamField>
<ParamField query="to_address_details" type="Json" >list of addresses used for receiving coins associated with the UTXO model; this value must be a JSON string and has a parameter type that fits</ParamField>
<ParamField query="fee" type="Float" >transaction fees per byte; applicable to UTXO model</ParamField>
<ParamField query="gas_price" type="Int" >gas price; applicable to ETH; unit: wei</ParamField>
<ParamField query="gas_limit" type="Int" >gas limit, applicable to ETH</ParamField>
<ParamField query="extra_parameters" type="Json" >calldata: request data associated with an EVM transaction, type: string; input_selection: inputs to be excluded and included in transactions associated with the UTXO model; inputs must be in correspondence with from_address (if specified); it must be a JSON string and has a structure that fits InputSelection</ParamField>

#### Response
<ResponseField name="fee_per_byte" type="Int" >transaction fees per byte; applicable to the UTXO model</ResponseField>
<ResponseField name="fee_amount" type="Int" >transaction fee; applicable to the UTXO model</ResponseField>
<ResponseField name="gas_price" type="Int" >ETH gas price</ResponseField> 
<ResponseField name="gas_limit" type="Int" >ETH gas limit</ResponseField>
